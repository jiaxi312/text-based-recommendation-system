### Required Packages
* tensorflow
* sklearn
* keras-resnet (Note: use the command to install)
`pip install git+https://github.com/vincenzodentamaro/keras-resnet.git`
* Pretrained word embedding matrix GloVe with 42 billion words in 300 dimensions.
